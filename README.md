# SRT editor

This is a experimental project to use the package [tview](http://github.com/rivo/tview).

![Demo](demo.gif)

## Features

* Increase/decrease time (ms)
* Edit subtitles
* Search for terms
* Convert file to UTF-8

## Next steps

* "Replace" and "Replace All"
* Revert from UTF-8 to other charsets
